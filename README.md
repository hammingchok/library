# Introduction

本项目是使用C++实现的基本网络库，采用Reactor事件分发模式，拟完善实现一个HTTP服务器。

  - C++ 11
  - Non-Blocking IO + IO multiplexing
  - Threadpool
  - Timer
  - Asynchronous log

# Envoirment
  - Ubuntu ~ 18.04
  - g++ 7.5


### Installation

```sh
$ cd src/version
$ make
```

### Usage

```sh
$ ./HttpServer [-p port]
```
